How to Install

Arrival Magazien Manager is really easy to install. Just follow the steps mentioned bellow.

Step1: Upload the magazine.zip file to your destination path where you want to install it.

Step2: Create a new database and username.

Step3: Import  the install.sql file to your database.

Step4: Edit .env file on your server root to update database information with your own.

 

Admin Panel

The defualt login url to login on admin panel is http://www.yoursite.com/admin (if you upload it on a subfolder then it will be http://www.yoursite.com/foldername/admin )

Default login Email: admin@admin.com

Default Password: admin

You can change the details from Admin Info section on your admin panel.

Add New Issue

Its very easy to add a new issue with the Arrival Magazine Manager.

To Add a new issue first login to your admin panel. From the Issue Manager go to  Add New Issue.

Add a name of the issue, cover and the pdf file of the issue. You can also add a description and tags about the new issue.
